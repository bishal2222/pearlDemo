### Project setup:

+ Clone the project via [Github Link](https://github.com/bishal2222/pearlDemo.git)   
+ Go to the project. For example: `cd cypress`
+  You will be in the master branch.
+ Run `npm install` , it will install all the prerequisite packages.

### Project run:

There are commands already defined in the package.json file to run the project. Below are the commands:
+ `npm run open-cypress` to run cypress spec file in cypress DOM
+ `npm run open-cypress:firefox`  to run cypress spec file in firefox
+ `npm run open-cypress:chrome` to run cypress spec file in chrome
